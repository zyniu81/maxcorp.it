# MAXCORP LCARS Terminal - Personal Portfolio

Welcome to my personal engineering terminal. This project is a fully functional, interactive web portfolio styled after the LCARS (Library Computer Access/Retrieval System) interface from Star Trek. It serves as a central hub for my professional profile, projects, and communication channels.

### 🚀 Core Specifications
* **Role:** Backend Systems, Data Analysis, and Architecture.
* **Objective:** Automating tedious tasks, solving complex logic puzzles, and building robust systems from the ground up.
* **Status:** Open to Junior and Freelancer opportunities.

### 🛠️ System Features
* **Dynamic GitHub Sensor Array:** Automatically fetches and analyzes data from my public repositories using the GitHub API, calculating language statistics on the fly.
* **Holographic Modals:** Custom CSS/JS animated interfaces for accessing personnel records (Bio) and project databases.
* **Bilingual Protocol:** Seamless, JavaScript-powered toggle between English and Polish localization.
* **Audio Feedback Subsystem:** Integrated, togglable sound effects for authentic LCARS interaction.
* **Responsive Design:** Fluid layout adapting perfectly to both desktop workstations and mobile comm-panels (iOS/Android).

### 💻 Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Custom variables, flexbox, media queries, and keyframe animations (No external libraries/frameworks).
* **Vanilla JavaScript (ES6+):** Asynchronous API calls, DOM manipulation, event delegation, and audio management.

### 📡 Deployment
The terminal is currently deployed and fully operational at: [maxcorp.it](https://maxcorp.it)

# MAXCORP LCARS Terminal - Portfolio Osobiste

Witaj w moim osobistym terminalu inżynieryjnym. Ten projekt to w pełni funkcjonalne, interaktywne portfolio internetowe stylizowane na interfejs LCARS (Library Computer Access/Retrieval System) ze Star Treka. Służy jako centralny węzeł mojego profilu zawodowego, projektów oraz kanałów komunikacji.

### 🚀 Specyfikacja Główna
* **Rola:** Systemy Backendowe, Analiza Danych i Architektura.
* **Cel:** Automatyzacja żmudnych zadań, rozwiązywanie zawiłych zagadek logicznych i budowanie solidnych systemów od zera.
* **Status:** Otwarty na zlecenia jako Freelancer oraz stanowiska typu Junior.

### 🛠️ Funkcje Systemu
* **Dynamiczny Skaner GitHuba:** Automatycznie pobiera i analizuje dane z moich publicznych repozytoriów za pomocą GitHub API, obliczając statystyki języków programowania w locie.
* **Holograficzne Modale:** Autorskie, animowane w CSS/JS interfejsy do przeglądania akt personelu (Bio) oraz bazy projektów.
* **Protokół Dwujęzyczny:** Płynne, oparte na JavaScript przełączanie między lokalizacją angielską i polską.
* **Podsystem Audio:** Zintegrowane, przełączalne efekty dźwiękowe dla autentycznej interakcji z LCARS.
* **Responsywność:** Elastyczny układ idealnie dopasowujący się zarówno do stacji roboczych, jak i urządzeń mobilnych (iOS/Android).

### 💻 Technologie
* **HTML5:** Struktura semantyczna.
* **CSS3:** Niestandardowe zmienne, flexbox, media queries i animacje klatkowe (Brak zewnętrznych bibliotek/frameworków).
* **Czysty JavaScript (ES6+):** Asynchroniczne wywołania API, manipulacja DOM, delegacja zdarzeń i zarządzanie audio.
